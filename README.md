# CupLife 🥤

**你的桌面情感伙伴 — 一杯有生命的水**

CupLife 是一款内置 AMOLED 屏幕和 AI 摄像头的智能水杯，通过可养成的虚拟宠物和 360° 旋转底座，将饮水提醒变成有温度的桌面陪伴体验。

## 核心特性

- 🐾 AI 虚拟宠物养成（4 种宠物 + 换装 / 成长 / 社交）
- 💧 智能饮水提醒（行为驱动，非冷冰冰的通知）
- 📷 主人识别 & 跟随（摄像头 + 旋转底座，始终面向你）
- 🧠 情绪感知 & 场景适配（专注守护、深夜陪伴、晨间问候等）

## 项目结构

- `ESP32-S3-Servo-BLE/` — 硬件固件（ESP32-S3 / FreeRTOS / LVGL）
- `iOS/` — iOS 客户端
