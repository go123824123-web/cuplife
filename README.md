# CupLife 🥤

**你的水杯里，住着一只小猫 🐱**

CupLife 是一款桌面情感伙伴水杯——通过 AI 摄像头、旋转底座和可养成的虚拟宠物，让喝水变得有温度。

## 四大场景

- 🐾 **认领你的水杯宠物** — 选一只专属伙伴，起名字，喂下第一口水唤醒它
- 💧 **日常饮水好搭子** — 催你喝水会赌气，晨间问候，深夜陪伴到凌晨
- 💻 **你的工作小伙伴** — 陪你专注工作，关注健康与情绪，温柔提醒别太累
- 🔄 **转身迎接小主人** — 舵机驱动 180° 旋转，追随你回到桌前

## 项目结构

- `ESP32-S3-Servo-BLE/` — 硬件固件
- `iOS/` — iOS 客户端
